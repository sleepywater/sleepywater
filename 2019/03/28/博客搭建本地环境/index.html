3、安装hexo-cli：进入终端，键入命令

其中 ‘cnpm’ 代替 npm，安装速度更快； ‘i’ 表示install， ‘-g’表示全局安装 (注意：如果安装失败，建议在管理者模式下安装)]

测试本地运行

上述环境完成后，我们就可以在本地预览blog了。很简单，两行命令。

  1. 键入命令 hexo generate 生成静态文件，可以简写为 hexo g
  2. 键入命令 hexo server 在本地运行， 可以简写为 hexo s
    出现下图则表示成功
